# Gen AI Equity Research Analyst

Prototype repository for a generative-AI powered equity research assistant.

## Project
- Purpose: Tools and notebooks for retrieving, analyzing, and summarizing equity research data using generative models.
- Key files: `main.py`, `requirements.txt`, and the `notebooks/` folder.

## Requirements
- Python 3.8+ (Windows)
- Install dependencies:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Usage
- Run the main script:

```powershell
python main.py
```

- Work with notebooks (open Jupyter):

```powershell
pip install jupyterlab
jupyter lab
```

Open files in the `notebooks/` folder to explore examples and experiments (e.g., `faiss.ipynb`, `retrival.ipynb`).

## Data
- Example data and notes are in `notebooks/` (e.g., `movies.csv`, `nvda_news_1.txt`, `sample_text.csv`).
